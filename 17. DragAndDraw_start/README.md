Drag And Draw - Starter Code
============================

Contains the starter code for the Drag And Draw app, taken
from Chapter 30 of the book
"[Android Programming: The Big Nerd Ranch 
Guide](https://bignerdranch.com/books/android-programming-the-big-nerd-ranch-guide-4th-edition/)",
4th Edition.

The code compiles and runs as is, but the app does nothing.
The following classes must be completed as per the TODOs: Box, BoxDrawingView.
